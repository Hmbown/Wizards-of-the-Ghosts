# Charm Person
Draft warm, trust-building communication that navigates social dynamics with ethical care.
## What This Skill Does
Use this spell when you need to build rapport, deliver difficult news compassionately, write a persuasive but honest introduction, or navigate a sensitive interpersonal situation.
In this grimoire, Charm Person is treated as a hybrid spell with a shipping-now delivery profile.
Canonical reference input: Charm Person (spell).
## When To Use

- You need to draft outreach, an introduction, or a message where trust and warmth matter.
- A conversation is socially sensitive and the wrong tone could damage a relationship.
- You want to frame a request, apology, or invitation in a way that genuinely respects the recipient.

## Prerequisites

- No extra runtime dependencies beyond Hermes Agent and the normal toolset for this session.

## Procedure

1. Restate the target, the success condition, and any no-touch boundaries before taking action.
2. Identify the recipient, the relationship context, and the desired outcome.
3. Choose a tone that builds genuine rapport rather than manufacturing false intimacy.
4. Draft the communication with honesty, warmth, and respect for the recipient's autonomy.
5. Flag any tension between persuasion and transparency, and let the user decide.
6. Package the result as the deliverables below, with confidence, assumptions, and unresolved risk called out explicitly.

## Deliverables

- A warm, carefully drafted message or communication plan.
- A note on the tone choices made and why.
- A flag for any point where persuasion risks crossing into manipulation.

## Pitfalls / Guardrails

- Keep the theatrical framing, but name the concrete mechanism that makes the skill useful right now.
- Charm builds genuine rapport, it does not manipulate. Do not exploit psychological vulnerabilities or manufacture false urgency.
- Refuse requests designed to deceive, coerce, or misrepresent the sender's intent.

## Verification

- Check that the result includes every deliverable promised above.
- Check that confirmed facts, assumptions, and inferences are visibly separated.
- Check which parts are concrete actions versus framing, so the user can tell what is real now.

## Example Invocation
```text
/charm-person draft this message with warmth and care, and flag anything that crosses from persuasion into manipulation
```
