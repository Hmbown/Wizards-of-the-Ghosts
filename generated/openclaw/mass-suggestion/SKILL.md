# Mass Suggestion

Craft a message that nudges an entire audience toward a single action.

## Overview

Mass Suggestion is interpreted here as a metaphorical spell with a shipping-now execution model.

Canonical source: Mass Suggestion (spell)

Provider target: OpenClaw

## When To Use

- You need to draft a company announcement, campaign message, or broadcast communication that drives a specific collective action.
- The audience is large enough that individual persuasion is impractical — you need one message that works at scale.

## Workflow

1. Define the single action you want the audience to take.
2. Identify the audience segments and what motivates each one.
3. Draft the message to work for the broadest segment while not alienating others.
4. Apply the manipulation audit at amplified scrutiny: at scale, even mild dark patterns cause real harm.
5. Return the message with deployment guidance and a note on which audience segments it may not reach.

## Deliverables

- The broadcast message, optimized for the target audience and action.
- Segment analysis: which groups will respond and which will not.
- An amplified manipulation audit: risks that emerge specifically because of scale.

## Guardrails

- Scaled influence requires scaled scrutiny. A nudge that is harmless one-on-one can become coercive when broadcast to thousands.
- Refuse propaganda patterns: emotional manipulation without factual basis, manufactured consensus, or suppression of dissent.

## Default Invocation

Use $mass-suggestion to draft a broadcast message that moves this audience toward this action, with extra scrutiny for scale-amplified manipulation.
