# Dimension Door

Jump cleanly between two known contexts without dragging the whole room.

## Overview

Dimension Door is interpreted here as a metaphorical spell with a shipping-now execution model.

Canonical source: Dimension Door (spell)

Provider target: OpenClaw

## When To Use

- You need a fast switch between two familiar branches, environments, accounts, or workspaces.
- The hard part is carrying the right context, not transforming the system itself.
- You want a reversible handoff rather than a permanent move.

## Workflow

1. Confirm the exact source and destination states, along with what must be carried across.
2. Package the minimum viable context for the jump: credentials, branch state, env vars, or notes.
3. Return the switch sequence plus a reentry checklist so you can step back without confusion.

## Deliverables

- A clean context-switch checklist.
- A list of state that must travel with you.
- A reentry note for getting back to the original location.

## Guardrails

- Do not use quick-switch assumptions for migrations, unknown environments, or destructive cutovers.
- Call out any hidden state that would be left behind and cause confusion on arrival.

## Default Invocation

Use $dimension-door to help me jump between these two known contexts with only the state I actually need to carry.
